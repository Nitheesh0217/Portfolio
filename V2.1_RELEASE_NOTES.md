## 🎮 Feature Breakdown

### 1. Smart Search (`lib/searchPortfolio.ts`)

**Capability:** Real-time full-text search across projects, certificates, and skills.

**Usage:**
```typescript
const results = searchAll(projects, certificates, metrics, "react");
// Returns: Array<SearchResult> sorted by relevance
```

**Relevance Algorithm:**
- Exact match: 1.0
- Starts with: 0.8
- Contains: 0.6
- No match: 0.0

**Where it's used:**
- `components/windows/SearchWindow.tsx` — UI component
- Can be used in any future search feature

**Performance:** O(n) per query, <10ms typical

---

### 2. Project Timeline (`components/windows/TimelineWindow.tsx`)

**Capability:** Chronological view of projects with ROI receipts.

**Visual Elements:**
- Timeline line with animated dots
- Project date (from `built_at` field)
- Tech stack badges (up to 4, +N for rest)
- ROI receipt with emerald styling
- Links to live URLs

**Sorting:** Newest first (projects without `built_at` excluded)

**Size:** 500×auto

**Pro tip:** Great for showing career progression during interviews.

---

### 3. Skills Visualization (`components/windows/SkillsWindow.tsx`)

**Capability:** Interactive tech stack network organized by category.

**Categories:**
- Frontend (React, Vue, Tailwind, etc.)
- Backend (Node.js, Python, Java, Go, etc.)
- Database (PostgreSQL, MongoDB, Redis, etc.)
- DevOps (Docker, Kubernetes, Vercel, AWS, etc.)
- AI (Claude, OpenAI, LangChain, RAG, etc.)
- Other (miscellaneous)

**Interactions:**
- Hover to see "Used in X projects"
- Opacity indicates hover state
- Color-coded by category

**Size:** 520×auto

**Pro tip:** Shows breadth of expertise at a glance.

---

### 4. Window State Persistence (`lib/windowStorage.ts`)

**Capability:** Remember window positions, sizes, and states.

**Storage:** localStorage under key `dws_window_state_v2`

**What's saved:**
- Position (x, y) for each window
- Open/closed state
- Minimized state
- Z-index order

**What's not saved:** Window content (refreshes each load)

**Size:** ~5KB typical

**How to integrate:**
```typescript
useEffect(() => {
  saveWindowState(windows); // Auto-save on changes
}, [windows]);
```

---

### 5. Resume Export (`lib/generateResume.ts`)

**Capability:** Download portfolio as markdown or JSON.

**Formats:**

**Markdown (.md):**
- Professional formatting
- Section headers
- Featured projects highlighted
- All certifications included
- Compact "All Projects" list

**JSON (.json):**
- Raw data export
- Machine-readable
- Timestamped export

**How to use:**
```typescript
const markdown = generateResumeMarkdown(name, email, projects, certs, metrics);
downloadResume(markdown, 'resume.md');
```

**Next step:** Feed markdown to Pandoc or a PDF library for PDFs.

---

### 6. Theme Customizer (`lib/themeCustomizer.ts`)

**Capability:** Swap between 5 themes or create custom colors.

**Built-in Themes:**
- Violet (default)
- Blue
- Emerald
- Orange
- Slate (minimal)

**Customization:**
```typescript
interface ThemeConfig {
  primaryColor: string;     // Main brand color
  accentColor: string;      // Secondary highlight
  backgroundDark: string;   // Page background
  borderColor: string;      // UI borders
  glassOpacity: number;     // 0-1 glass effect
  useGradient: boolean;     // Gradient backgrounds
}
```

**Storage:** localStorage under key `dws_theme_config_v1`

**How to use:**
```typescript
saveTheme(PRESET_THEMES.blue); // Apply preset
// or
saveTheme(customConfig);        // Apply custom
```

---

### 7. Advanced AI Prompt (`lib/assistantAdvancedPrompt.ts`)

**Capability:** Multi-mode Claude reasoning with deeper context.

**Modes:**
1. **Hiring Manager Brief** (default)
   - Fast, credible signals
   - 30-second response
   - Specific metrics

2. **Technical Deep Dive**
   - Architecture reasoning
   - Tradeoff analysis
   - Failure mode discussion

3. **Growth & Learning**
   - Vulnerability + lessons
   - Real failures → outcomes
   - Career reflection

**Anti-hallucination:**
- Rejects unknown projects
- Admits knowledge gaps
- Pivots to related evidence
- Never invents metrics

---

### 8. Window Resizing Hook (`hooks/useWindowResize.ts`)

**Capability:** Drag corners/edges to resize windows dynamically.

**Constraints:**
- Min width: 300px
- Min height: 200px
- Real-time feedback

**How to integrate into SystemWindow.tsx:**
```typescript
const { resizeRef, handleMouseDown } = useWindowResize(onResize);

return (
  <div {...}>
    {/* content */}
    <div ref={resizeRef} onMouseDown={handleMouseDown} className="resize-handle" />
  </div>
);
```
